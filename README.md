# 舟券アカデミア評価ツール v63 Androidコピペ強化版

## 既存のVercelサイトを更新する場合
1. このZIPを解凍します。
2. `hunaken-academia-v63-android-compressed-paste-fix` フォルダの中身を全部選択します。
3. GitHubの既存リポジトリ `hunaken_academia` にアップロードします。
4. `Commit changes` を押します。
5. Vercelが自動で再デプロイします。

## Vercel設定
- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

## ローカル確認
```bash
npm install
npm run dev
```

## 今回の変更
- 上部の「場平均で見る1着率」表示を削除しました。
- 総合AI評価内の表示を「場平均＋風」に変更しました。
- 「場平均＋風」は `場別・季節別の基本1着率 + 風補正` で表示します。
- AI点数側では、枠基準に風を混ぜず、風は「風」項目で1回だけ反映するようにして、風補正の二重反映を防ぎました。
- 展示タイム＋一周＋まわり足の補正は内部計算に残しています。
- 自動取得など、V100以降の追加機能は入れていません。

## v6.3.6 Vercel npm install対策

- package-lock.jsonを削除しました。
- 以前のlockには生成環境の内部npmレジストリURLが混ざる可能性があったため、Vercelでnpm installが失敗する原因になり得ました。
- .npmrcで public npm registry を明示しました。
- Node.jsを20.xに固定しました。
